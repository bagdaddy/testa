## Goal

Apply the variation's `changes[]` array — CSS, HTML, text, attribute, JS injection — to the live DOM. Idempotent across re-eval cycles (SPA route changes). Late-arrival selectors handled via MutationObserver.

## Acceptance criteria

- `applyChanges(variation: Variation): void` iterates `changes[]` in order.
- `type: 'css'` injects a `<style>` tag (or extends an existing one) with the supplied selector + styles. Idempotent via stable IDs (`data-testa-css-${variationId}`).
- `type: 'html'` replaces innerHTML of matching elements with the supplied HTML. Sanitizes against script tags (DOMParser parse + serialize). Customers wanting JS injection use `type: 'js'`.
- `type: 'text'` replaces textContent.
- `type: 'attribute'` sets `element.setAttribute(name, value)`.
- `type: 'js'` evaluates the JS string in a `new Function(code)()` context; errors caught and logged to `_pixel_health`.
- For all DOM-targeting types: if the selector matches no current elements, register a MutationObserver that retries when elements appear. Polling fallback every `CLICK_SELECTOR_TIMEOUT` ms up to `CLICK_SELECTOR_MAX_TRIES` times for legacy compat.
- Fires `'variation_applied'` event via `eventEmitter` after all changes apply (legacy compat with 3.x).
- Re-eval (SPA route change) re-applies — but skips no-op when the same variation already has its CSS / HTML / text in place.

## Implementation notes

```ts
// apps/pixel/src/runtime/experiments/apply/index.ts (sketch)
const APPLIERS: Record<string, (change: any) => void> = {
  css: applyCss,
  html: applyHtml,
  text: applyText,
  attribute: applyAttribute,
  js: applyJs,
};

export function applyChanges(variation: Variation): void {
  for (const change of variation.changes) {
    const fn = APPLIERS[change.type];
    if (!fn) continue;
    try { fn(change); } catch (err) { health.recordError('apply', err); }
  }
  fireEvent('variation_applied', { variation_id: variation.variation_id });
}
```

For MutationObserver-based late-arrival:

```ts
function whenElementMatches(selector: string, fn: (el: Element) => void): void {
  const initial = document.querySelectorAll(selector);
  if (initial.length > 0) { initial.forEach(fn); return; }
  const obs = new MutationObserver(() => {
    const els = document.querySelectorAll(selector);
    if (els.length > 0) { els.forEach(fn); obs.disconnect(); }
  });
  obs.observe(document.body, { childList: true, subtree: true });
  // Bounded: disconnect after N seconds.
}
```
