## Goal

Durable, retry-capable event transport. Every event is written to an IndexedDB outbox first, then attempted via `fetch keepalive` POST `/track`. Retries on 5xx with exp backoff; force-flushes via `sendBeacon` on `pagehide`. Surface delivery health via the `_pixel_health` synthetic event.

This is the linchpin of "tracking reliability" — without it, customers on flaky networks lose events silently.

## Context

- `event_id` is UUIDv7, generated client-side, persisted in the outbox. Same event retried any number of times → same UUID → dedup at the collector's Redis Stream layer.
- Outbox is bounded (~500 events × ~1 KB = 500 KB) with FIFO eviction.
- `_pixel_health` synthetic event is reserved (see `event-shape.md`).

## Acceptance criteria

### `outbox.ts`
- `enqueue(event)` writes to IDB synchronously-feeling (return after the write is queued, not after CH ack).
- `pending()` returns up to N oldest events.
- `markSent(eventIds)` deletes those from IDB.
- `count()` returns total outbox size.
- `oldestAgeMs()` returns the age of the oldest entry (for `_pixel_health`).
- FIFO eviction at 500 entries.
- localStorage fallback when IDB is unavailable (Safari Private Mode pre-2022). Same API; smaller capacity.

### `uuid7.ts`
- `uuidv7()` returns a 36-char canonical UUIDv7 string. Monotonic per-millisecond (within a tab).

### `transport.ts`
- `flush()` reads pending events, POSTs `/track` with `fetch(..., { keepalive: true })`.
- Response 2xx → `markSent`.
- Response 5xx → retry with exp backoff 200 ms → 30 s + jitter.
- Response 4xx → log + drop (poison; mark sent so we stop retrying).
- Network error → treat as 5xx (retry).
- `pagehide` and `visibilitychange:hidden` listeners trigger an immediate flush via `sendBeacon` fallback.
- Auto-flush every 500 ms or on every 10th event, whichever first.
- On runtime init: drain leftover IDB entries before sending fresh ones.

### `health.ts`
- Emits `_pixel_health` synthetic event ~hourly per visitor with: `queued`, `sent`, `dropped`, `retried`, `oldest_age_ms`. Counts maintained in module-local accumulators.

### Tests
- Outbox round-trip: enqueue → pending → markSent.
- FIFO eviction at capacity.
- localStorage fallback when IDB unavailable (mock by deleting `window.indexedDB`).
- Transport: 2xx flow happy path.
- Transport: 5xx → retry with backoff (mock fetch to return 503 then 200).
- Transport: 4xx → drop (single attempt, then markSent).
- pagehide event triggers a synchronous flush via `sendBeacon`.
- UUIDv7: monotonic within a millisecond, valid format (`/^[0-9a-f]{8}-[0-9a-f]{4}-7[0-9a-f]{3}-[0-9a-b][0-9a-f]{3}-[0-9a-f]{12}$/`).
- `_pixel_health` is emitted with the right counts after a flush cycle.

## Implementation notes

UUIDv7 (per RFC 9562): 48-bit unix-millis timestamp + 4-bit version (`0b0111`) + 12-bit random + 2-bit variant + 62-bit random. Implementation references: `https://www.rfc-editor.org/rfc/rfc9562.html#section-5.7`.

```ts
// apps/pixel/src/runtime/network/transport.ts (sketch)
let backoffMs = 200;
const MAX_BACKOFF_MS = 30_000;

async function flush(): Promise<void> {
  const events = await outbox.pending();
  if (events.length === 0) return;

  try {
    const res = await fetch(config.endpoint, {
      method: 'POST',
      keepalive: true,
      headers: { 'content-type': 'application/json' },
      body: JSON.stringify({ events }),
    });
    if (res.ok) {
      await outbox.markSent(events.map((e) => e.event_id));
      backoffMs = 200; // reset
    } else if (res.status >= 500) {
      schedule(backoffMs);
      backoffMs = Math.min(backoffMs * 2 + jitter(), MAX_BACKOFF_MS);
      health.recordRetry(events.length);
    } else {
      // 4xx — poison; drop
      await outbox.markSent(events.map((e) => e.event_id));
      health.recordDropped(events.length);
    }
  } catch (err) {
    // network error → treat as 5xx
    schedule(backoffMs);
    backoffMs = Math.min(backoffMs * 2 + jitter(), MAX_BACKOFF_MS);
    health.recordRetry(events.length);
  }
}
```

`pagehide` flush via `sendBeacon`:

```ts
window.addEventListener('pagehide', () => {
  outbox.pending().then((events) => {
    if (events.length === 0) return;
    navigator.sendBeacon(config.endpoint, JSON.stringify({ events }));
    // Optimistic markSent — if beacon failed we'll retry on next pageload
    outbox.markSent(events.map((e) => e.event_id));
  });
});
```
