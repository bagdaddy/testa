## Goal

`GET /projects/:slug.js` reads the project's config + matching integration bundle from KV, returns a single combined JS response with `window.cfPrefill = {...}` injected, a loader stub inlined, and a `<script>`-loader pointing at the runtime URL (or an inlined runtime for v1 simplicity — pick inlined).

## Acceptance criteria

- Reads `project_config:{slug}` from `KV_PROJECT_CONFIG`. 404 if missing.
- Reads `integration_bundle:{integration_version}` from `KV_INTEGRATION_BUNDLES`. The bundle is the precompiled `loader.min.js + runtime.min.js` text, uploaded by Phase 5 backfill.
- Returns:
  ```
  Content-Type: application/javascript; charset=utf-8
  Cache-Control: public, max-age=60, stale-while-revalidate=300
  ETag: "<config_hash>"
  Set-Cookie: _testa_uuid=...; Domain=...; Max-Age=63072000; ...
  ```
- Body shape:
  ```js
  // 1. cfPrefill assignment
  window.cfPrefill = { project: <project_config_json>, env: 'production' };
  // 2. loader.min.js inlined (verbatim from KV)
  // 3. runtime.min.js inlined (verbatim from KV)
  ```
- If `If-None-Match` matches `config_hash`, return 304.
- If `integration_version` is `3.4` or `3.6`, return the legacy bundle as-is (no `cfPrefill` injection needed for legacy — they read `window.crbData` etc. set elsewhere). The legacy bundles have their own bootstrap.

## Implementation notes

KV uploads:

- `KV_PROJECT_CONFIG` holds `project_config:{slug}` — JSON. Populated by crobot's `PublishProjectConfigToKV` job (Phase 5.4).
- `KV_INTEGRATION_BUNDLES` holds three keys: `integration_bundle:3.4`, `integration_bundle:3.6`, `integration_bundle:4.0`. Uploaded once per release. Phase 0.x bootstrap or a manual `wrangler kv:key put` on first deploy.

```ts
// apps/edge/src/routes/serve.ts (final shape)
import { Hono } from 'hono';
import { renderPixel } from '../serve/render.ts';
import { getOrCreateVisitorId } from '../cookies.ts';

export const serveRoutes = new Hono<{ Bindings: Env }>();

serveRoutes.get('/projects/:slug{.+\\.js}', async (c) => {
  const slug = c.req.param('slug').replace(/\.js$/, '');
  const configRaw = await c.env.KV_PROJECT_CONFIG.get(`project_config:${slug}`);
  if (!configRaw) return c.text('// project not found\n', 404);
  const config = JSON.parse(configRaw);

  const bundle = await c.env.KV_INTEGRATION_BUNDLES.get(
    `integration_bundle:${config.integration_version}`,
  );
  if (!bundle) return c.text('// integration bundle missing\n', 500);

  // ETag
  const ifNone = c.req.header('if-none-match');
  if (ifNone && ifNone.replace(/"/g, '') === config.config_hash) {
    return new Response(null, { status: 304 });
  }

  const visitor = await getOrCreateVisitorId(c.req.raw, c.env);
  const body = renderPixel(config, bundle, c.env.ENVIRONMENT);

  return new Response(body, {
    status: 200,
    headers: {
      'content-type': 'application/javascript; charset=utf-8',
      'cache-control': 'public, max-age=60, stale-while-revalidate=300',
      etag: `"${config.config_hash}"`,
      'set-cookie': visitor.set_cookie_header,
    },
  });
});
```

`render.ts`:

```ts
// apps/edge/src/serve/render.ts
import type { ProjectConfig } from '@testa-platform/shared-types';

export function renderPixel(config: ProjectConfig, bundle: string, env: string): string {
  if (config.integration_version === '3.4' || config.integration_version === '3.6') {
    // Legacy bundles include their own bootstrap; just serve verbatim.
    return bundle;
  }
  // 4.0: prefill + bundle
  return [
    `(function(){`,
    `  window.cfPrefill = window.cfPrefill || {};`,
    `  window.cfPrefill.project = ${JSON.stringify(config)};`,
    `  window.cfPrefill.env = ${JSON.stringify(env)};`,
    `})();`,
    bundle,
  ].join('\n');
}
```
