## Goal

The actual ingest forwarder. `forwardBatch(events, env)` builds the `IngestBatch`, HMAC-signs it, POSTs to `${env.INGEST_ORIGIN_URL}/_ingest`, retries on 5xx, gives up on 4xx (with loud log).

## Acceptance criteria

- HMAC-SHA256 of the request body (covers `signed_at` and `events`) using `crypto.subtle` (Worker-native).
- Header `X-Edge-Signature: <hex>`.
- POST to `${env.INGEST_ORIGIN_URL}/_ingest` with `content-type: application/json`.
- On 5xx: throw to caller (DurableObject's flush retry handles it).
- On 4xx: log loudly, do NOT throw (poison batch — don't loop on it). Drop the events.
- On 2xx: return cleanly.
- Verifies against the `auth/hmac.ts` collector helper end-to-end (use a known test vector).
- Phase 2.5's `flushFn` stub is replaced with this function.

## Implementation notes

```ts
// apps/edge/src/ingest.ts (sketch)
import type { EnrichedEvent, IngestBatch } from '@testa-platform/shared-types';

export async function forwardBatch(events: EnrichedEvent[], env: Env): Promise<void> {
  const batch: IngestBatch = { signed_at: Date.now(), events };
  const body = JSON.stringify(batch);
  const sig = await sign(body, env.INGEST_SHARED_SECRET);

  const res = await fetch(`${env.INGEST_ORIGIN_URL}/_ingest`, {
    method: 'POST',
    headers: {
      'content-type': 'application/json',
      'x-edge-signature': sig,
    },
    body,
  });

  if (res.status >= 500) {
    throw new Error(`collector ${res.status}`);
  }
  if (res.status >= 400) {
    console.error('[ingest] poison batch dropped', res.status, await res.text());
    return;
  }
  // 204 / 200 — success
}

async function sign(body: string, secret: string): Promise<string> {
  const key = await crypto.subtle.importKey(
    'raw',
    new TextEncoder().encode(secret),
    { name: 'HMAC', hash: 'SHA-256' },
    false,
    ['sign'],
  );
  const sig = await crypto.subtle.sign('HMAC', key, new TextEncoder().encode(body));
  return [...new Uint8Array(sig)].map((b) => b.toString(16).padStart(2, '0')).join('');
}
```
